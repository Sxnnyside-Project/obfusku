## What is Obfusku?

Obfusku is a **symbolic programming language** where meaning is carried through glyphs rather than keywords.

It is:
- **Symbol-first**: Operations are invoked through Unicode symbols
- **Stack-based**: Execution uses a virtual machine with stack semantics
- **Ritual-oriented**: Programs are "spells" that must be properly sealed
